# MonsoonFlow Coordinator 🌧️

A smart daily schedule manager designed for Indian users facing monsoon disruptions like rain and traffic. Rebuilt from the ground up using **Vanilla JavaScript, HTML, and CSS** for maximum performance and a premium aesthetic.

## ✨ Features

- **AI ScheduleAgent**: Intelligent rescheduling of outdoor tasks during rain based on historical patterns (e.g., "MTH Road traffic" or "Market early closures").
- **Dynamic Rain Background**: A custom canvas-based rain engine that intensifies when "Rainy Mode" is enabled.
- **Glassmorphic UI**: A premium, modern dark-themed interface with smooth transitions and responsive layout.
- **Smart Suggestions**: AI-driven recommendations for working from home, choosing indoor alternatives, and avoiding traffic hotspots.
- **Productivity Insights**: Tracks time saved and tasks adjusted to give you a clear picture of your rain-ready efficiency.

## 🚀 Getting Started

Since this is a Vanilla JS project, you have two ways to run it:

1. **Recommended**: Run the development server:
   ```bash
   npm start
   ```
   Then open the local URL (usually `http://localhost:3000`).

2. **Direct**: Simply open the `index.html` file in any modern web browser.

## 🛠️ Technology Stack

- **Frontend**: Vanilla JavaScript (ES6+), HTML5, CSS3 (Custom Design System).
- **Icons**: Lucide Icons.
- **Typography**: Google Fonts (Outfit & Inter).
- **Core Engine**: `agent.js` (Scheduling Intelligence) & `app.js` (UI Orchestration).

## 🧠 AI Intelligence

The `ScheduleAgent` utilizes a repository of historical user notes and current weather status to optimize your day:
- Moves outdoor tasks to safer time slots.
- Suggests indoor alternatives for low-priority outdoor items.
- Provides context-aware reasoning for every adjustment.
